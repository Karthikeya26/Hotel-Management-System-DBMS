STRING FUNCTIONS

1.     Ascii() - Returns numeric value of left-most character

2.     bin() - Returns a string representation of the argument

3.     bit_length() - Returns length of argument in bits

4.     char_length() - Returns number of characters in argument

5.     Concat() - Returns concatenated string.

6.     Concat_ws() - Returns concatenate with separator.

7.     Elt() - Returns string at index number.

8.     Field() - Returns the index (position) of the first argument in the subsequent arguments.

9.     Format() - Returns a number formatted to specified number of decimal places

10.                        Hex() - Returns a string representation of a hex value.

11.                        Insert() - Inserts a substring at the specified position up to the specified number of characters

12.                        Instr() - Returns the index of the first occurrence of substring

13.                        Left() - Returns the leftmost number of characters as specified

14.                        Length() - Returns the length of a string in bytes

15.                        Locate() -  Returns the position of the first occurrence of substring

16.                        Lcase() - Synonym for LOWER() – converts to lowercase.

17.                        Lpad() - Returns the string argument, left-padded with the specified string

18.                        Ltrim() - Removes leading spaces

19.                        Mid() - Returns a substring starting from the specified position

20.                        Oct() - Returns a string representation of the octal argument

21.                        Position() - A synonym for LOCATE() -- Returns the position of the first occurrence of substring

22.                        Repeat() - Repeats a string the specified number of times

23.                        Replace() - Replaces occurrences of a specified string

24.                        Right() - Returns the specified rightmost number of characters

25.                        Rpad() - Appends string the specified number of times

26.                        Rtrim() - Removes trailing spaces

27.                        Space() - Returns a string of the specified number of spaces

28.                        Strcmp() - Compares two strings

29.                        Trim() - Removes leading and trailing spaces

30.                        Usecase() - Synonym for UPPER() -- Converts to uppercase.
