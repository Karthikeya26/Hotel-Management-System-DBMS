DATE FUNCTIONS

1.     Date() - The date() is a MySQL date/time function. It is used to get the date from given date/datetime.

2.     Adddate() - The adddate() is a MySQL date/time function. It is used to get the date in which some time/date intervals are added.

3.     Curdate() - The curdate() is a my date/time function. It is used to get the current date.

4.     Date_format() - The date_format() is a MySQL date/time function. It is used to get the date in specified format.

5.     Datediff() - The datediff() is a MySQL date/time function. It is used to get the difference between two date values.

6.     Day() - The day() is a MySQL date/time function. It is used to get the day from the given date.

7.     Dayname() - The dayname() is a MySQL date/time function. It is used to get the name of the day from the given date.

8.     Dayofmonth() - The dayofmonth() is a MySQL date/time function. It is used to get the day from the given date.

9.     Dayofweek() - The dayofweek() is a MySQL date/time function. It is used to get the day of the week in numeric.

10.                        Dayofyear() - The dayofyear() is a MySQL date/time function. It is used to get the number of day of the year.

11.                        From_days() - The from_days() is a MySQL date/time function. It is used to get the date of the given number of days.

12.                         Hour() - The from_days() is a MySQL date/time function. It is used to get the hour from the given datetime.

13.                         Addtime() - The addtime() is a MySQL date/time function. It is used to get time/datetime value in which some time intervals are added.

14.                        Current_timestamp() - The current_timestamp() is a MySQL date/time function. It is used to get the current date and time.

15.                        Current_time() - The current_time() is a MySQL date/time function. It is used to get the current time.

16.                        Last_day() - The last_day() is a MySQL date/time function that accepts date or datetime value only. It is used to get the last day of the given month for the input date. If we provide an invalid input date or datetime, it always returns a NULL value.

17.                        Localtime() - The localtime() is a MySQL date/time function. It is used to get the current date and time.

18.                        Localtimestamp()-The localtimestamp() is a MySQL date/time function. It is used to get the current date and time.

19.                        Makedate() - The makedate() is a MySQL date/time function. It is used to make the date of the given year and number of days.

20.                        Maketime() - The maketime() is a MySQL date/time function. It is used to make the time from given hour, minute and second.

21.                        Microsecond() - The microsecond() is a MySQL date/time function. It is used to get the value of microsecond from given datetime or time.

22.                        Minute() - The minute() is a MySQL date/time function. It is used to get the value of minute from given datetime or time.

23.                        Month() - The month() is a MySQL date/time function. It is used to get the value of month from given datetime or time.

24.                        Monthname() - The monthname() is a MySQL date/time function. It is used to get the full month name.

25.                        Now() - The now() is a MySQL date/time function. It is used to get the current date and time.

26.                        Period_add() - The period_add() is a MySQL date/time function. This function adds the given number of month in the given period in the formate YYMM or YYYYMM.

27.                        Period_diff() - The period_diff() is a MySQL date/time function. It is used to get the difference between two given periods.

28.                        Sec_to_time() - The sec_to_time() is a MySQL date/time function. It is used to convert the given second into time.

29.                        Str_to_date() - The str_to_date() is a MySQL date/time function. It is used to convert the string into given format_mask.

30.                        Subdate() - The subtime() is a MySQL date/time function. It is used to get the time/datetime which is subtracted by certain intervals.

31.                        Sysdate() - The sysdate() is a MySQL date/time function. It is used to get the system date.

32.                        Time() - The time() is a MySQL date/time function. It is used to get the time from given time/datetime.

33.                        Time_format() - The time_format() is a MySQL date/time function. It is used to format the time in specified given format_mask.

34.                        Time_to_sec() - The time_to_sec() is a MySQL date/time function. It is used to convert the time into seconds.

35.                        Timediff() - The timediff() is a MySQL date/time function. It is used to get the difference from given two time/datetime.

36.                        To_days() - The to_days() is a MySQL date/time function. It is used to convert the date into numeric number of days.

37.                        Timestamp() - The timestamp() is a MySQL date/time function. It is used to convert the expression into datetime/time.

38.                        Weekday() - The weekday() is a MySQL date/time function. It is used to get the index for a date.

39.                        Week() - The week() is a MySQL date/time function. It is used to get the week portion from given date.

40.                        Weekofyear() - The weekofyear() is a MySQL date/time function. It is used to get the week of the given date.
