MATH FUNCTIONS

1.     Abs() - The abs() is a Math function of MySQL. This function is used to get the absolute value of the given number.

2.     Acos() – The acos() is a Math function of MySQL. This function is used to get the arc cosine of the given number.

3.     Sign() - The sign() is a Math function of MySQL. This function is used to get a sign of the given number. If the number is less than 0, this function returns -1. If the number is 0 or 1 or greater than 0, this function returns 0.

4.     Sin() - The sin() is a Math function of MySQL. This function is used to get the sine value of the given number.

5.     Sqrt() - The sqrt() is a Math function of MySQL. This function is used to get the square root of the given number.

6.     Sum() - The sum() is a Math function of MySQL. This function is used to sum the values of given expressions.

7.     Tan() - The tan() is a Math function of MySQL. This function is used to get the tangent of the given number.

8.     Truncate() - The truncate() is a Math function of MySQL. This function is used to get the given number truncated up to given certain decimal places.

9.     Asin() - The asin() is a Math function of MySQL. This function is used to get the arc sine of the given number.

10.                         Atan2() - The atan2() is a Math function of MySQL. This function is used to get the arc tangent of the given number n and m.

11.                         Atan() - The atan() is a Math function of MySQL. This function is used to get the arc tangent of the given number.

12.                         Avg() - The avg() is a Math function of MySQL. This function is used to get the average value from the given expression.

13.                         Ceil() - The ceil() is a Math function of MySQL. This function returns the smallest value which is greater than or equal to the given number.

14.                         Cos() - The cos() is a Math function of MySQL. This function is used to get the cosine of the given number.

15.                         Cot() - The cot() is a Math function of MySQL. This function is used to get the cotangent of the given number.

16.                         Count() - The count() is a Math function of MySQL. This function is used to get the total count for a particular column of the table.

17.                         Div() - The div() is a Math function of MySQL. This function is used for integer division, in which n is the number to be divided by the number m.

18.                        Exp() - The exp() is a Math function of MySQL. This function is used to find e raised to the power of number i.e. enumber.

19.                         Floor() - The floor() is a Math function of MySQL. This function is used to find the greatest integer which is equal to or less than the given number.

20.                         Greatest() - The greatest() is a Math function of MySQL. This function is used to get the largest number from the list.

21.                         Least() – The least() is a Math function of MySQL. This function is used to get the smallest number from the list.

22.                         Ln() - The LN() is a Math function of MySQL. This function is used to get the natural logarithm of the given number.

23.                         Log10() - The Log10() is a Math function of MySQL. This function is used to get the base-10 logarithm of given number.

24.                         Log()-The Log() is a Math function of MySQL. This function is used to return either the natural logarithm of the given number if it has one parameter or specified base logarithm number if it has two parameters.

25.                        Log2() - The log2() is a Math function of MySQL. This function is used to get the base-2 logarithm of given number.

26.                         Max() - The max() is a Math function of MySQL. This function is used to get the maximum number of the given column name.

27.                        Mod() - The mod() is a Math function of MySQL. This function is used to get the remainder of given values.

28.                        Power() - The power() is a Math function of MySQL. This function is used to get the power of the given values.

29.                        Radians() - The radians() is a Math function of MySQL. This function is used to convert the given degrees to radians.

30.                        Rand() - The rand() is a Math function of MySQL. This function is used to generate the random number.

31.                         Round() - The round() is a Math function of MySQL. This function is used to round off the given number.
